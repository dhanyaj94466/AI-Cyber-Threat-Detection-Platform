## 📌 Project Overview

The **AI Cyber Threat Detection Platform** is a web-based cybersecurity application developed as a **Bachelor of Computer Applications (BCA) Final Year Project**.

It enables security analysts to analyze log files, detect cyber threats, scan websites, monitor security events, generate AI-inspired security recommendations, and export professional reports.

# 🤖 AI Decision Engine

The AI Decision Engine analyzes:

- Critical Threats
- High Threats
- Medium Threats
- Low Threats
- Failed Login Attempts
- Security Score

Based on these values, it determines:

- Threat Level
- AI Confidence
- Risk Assessment
- Recommended Security Actions

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/dhanyaj94466/AI-Cyber-Threat-Detection-Platform.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Start Backend

```bash
uvicorn main:app --reload
```

## Run Frontend

Open:

```
Frontend/index.html
```
